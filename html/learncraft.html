LearnCraft Documentation

Welcome to the documentation for the LearnCraft project. This project is designed to help study in a gamified environment.

Installation

To install LearnCraft, follow these steps:

  1. Clone the repository: git clone https://github.com/m4lw4re4rt/learncraft.git
  2. Navigate to the project directory: cd learncraft
  3. Install the required dependencies: pip install -r requirements.txt
